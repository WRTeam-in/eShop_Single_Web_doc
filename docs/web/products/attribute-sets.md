# Attribute Sets Management

Attribute Sets allow you to **group related attributes together** into reusable collections.  
By assigning an Attribute Set to a product, you can automatically load all relevant attributes (e.g., a "T-Shirt" set might include Size, Color, and Material), streamlining the product creation process.

The **Attribute Sets** module enables you to create and manage these collections efficiently.

![Attribute Sets Page Overview](/img/web/attribut_sets_tab.jpg)

---

## Overview

The Attribute Sets page facilitates:
- **Creating new sets** for different product categories
- **Managing existing sets** via a centralized list
- **Enabling or Disabling sets** to control their availability

---

## Create New Attribute Set

To define a new group of attributes, click the **Add Attribute Set** button. This will open the configuration interface.

![New Attribute Set Form](/img/web/attribut_sets3_tab.jpg)

### Configuration Fields

#### Attribute Set Name
Enter a unique name for the set that clearly describes the category of products it applies to.

**Examples:**
- `Smartphone Specs`
- `Men's Clothing`
- `Furniture Dimensions`

:::info
Choose names that correspond to your product categories to make assignment intuitive for vendors or admins.
:::

### Actions

- **Reset**  
  Clears the input field.

- **Add Attribute Set**  
  Saves the new set and makes it available for attributes to be assigned to it.

---

## Attribute Sets List

The main table displays all available attribute sets and provides tools for management.

### Control Toolbar

- **Search** – Find sets by name
- **Refresh** – Reload the list details
- **View Options** – Customize column visibility
- **Export** – Download the list

### Data Columns

| Column | Description |
|------|-------------|
| **ID** | Unique system identifier |
| **Name** | Display name of the attribute set |
| **Status** | Toggle to Enable (Active) or Disable (Inactive) the set |
| **Action** | Options to Edit or Delete the set |

### Set Actions

From the **Action** column, you can:

- **Edit**  
  Rename the attribute set.

- **Delete**  
  Remove the attribute set from the system.

:::caution
Before deleting an attribute set, ensure it is not critically required by existing products or attributes.
:::
