## Languages Overview

<!-- ![Languages](/img/web/lanuage.png) -->

This page is used to **add languages, manage labels, and control language behavior**.

---

## Language Management

- **Add Language** – Create a new language
- **Language Dropdown** – Select a language to edit its labels
- **Enable RTL** – Enable right-to-left layout for supported languages

---

## Labels Configuration

This section allows you to update **text labels shown to users**.

Common labels include:

- **Menu**
- **Products**
- **Product**
- **My Account**
- **My Orders**
- **Favorite**
- **Login**
- **Register**
- **About Us**
- **Contact Us**

Each input field controls the **frontend display text** for the selected language.

---

## Language Behavior

- Labels are applied based on the selected language
- Changes reflect immediately after saving
- RTL layout applies automatically when enabled

---

## Actions

- **Save / Update** – Apply language label changes
- **Add Language** – Create a new language
- **Select Language** – Switch between available languages
